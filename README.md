# Learn Rust: Bit Manipulator

A simple interactive terminal-based application to manipulate and visualize a 16-bit integer.

## Features

*   Set a bit to 1 (high)
*   Set a bit to 0 (low)
*   Toggle a bit
*   Read a bit
*   Displays the value in binary, hexadecimal, and decimal.

## How to Run

```bash
cargo run --release
```
